# Acid ManuForm

#### "Manu-" = "Hand"
#### "-Form" = "Shape"

The Acid ManuForm is a 40% split ergo keyboard with **rj11** connectors and **Raspberry Pi Pico**.

# Components
|Component Name|Number Needed|
|-|-|
|Raspberry Pi Pico|2|
|rj11 female connectors|2|
|Keyboard Switch|42|
|1N4148 Diode|42|
|Wire| However much you need|
|Set of Keycaps|1|
|rj11 cable|1|
|micro usb cable|1|
